=== Romangie Theme ===

Romangie is a responsive wordpress theme. It is kept simple and supports all wordpress post formats.

== Features ==

*	Romangie is built with the bootstrap framework, so it is highly flexible and looks great on smartphones, tablets and desktops.
*	Different widget areas
*	Supports Wordpress Post formats
*	2 Pages Themes for different purposes

== Installation ==

= Manual installation =

1. Upload the 'romangie' folder to the `/wp-content/themes/` directory

= Using the Wordpress Admin Center =

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for ‘pinboard’
3. Click the ‘Install’ button to open the theme’s repository listing
4. Click the ‘Install’ button
